\## IRL Stream Widget (OBS)



Simple HTML widget for IRL streams:

\- local time based on city

\- temperature

\- country flag

\- manual location change via `location.txt`



\### Setup

1\. Get free API key from https://openweathermap.org

2\. Paste it into `script.js`

3\. Add `index.html` as Browser Source in OBS



\### Change location

Edit `location.txt` and refresh Browser Source.
